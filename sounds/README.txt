CÁC FILE ÂM THANH
===============

Game Tetris yêu cầu các file âm thanh sau để có hiệu ứng âm thanh:

1. move.mp3 - Phát khi di chuyển khối sang trái/phải
2. rotate.mp3 - Phát khi xoay khối
3. drop.mp3 - Phát khi khối chạm đáy
4. clear.mp3 - Phát khi xóa 1-3 dòng
5. tetris.mp3 - Phát khi xóa 4 dòng (Tetris)
6. level-up.mp3 - Phát khi tăng cấp độ
7. game-over.mp3 - Phát khi game kết thúc

HƯỚNG DẪN
---------

Bạn có thể tải các file âm thanh từ internet và đặt vào thư mục này. 
Các định dạng âm thanh hỗ trợ bởi hầu hết các trình duyệt là:
- MP3
- WAV
- OGG

LƯU Ý: Nếu bạn không có các file âm thanh, game vẫn sẽ hoạt động bình thường nhưng sẽ không có âm thanh.
